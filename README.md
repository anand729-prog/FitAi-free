## 🚀 Getting Started

### Step 1 — Open the app

Download `FitAI-Free.html` and **double-click** it in your browser.
That's it — no installation, no build step, no server needed.

### Step 2 — Get your FREE Gemini API Key

1. Go to **[aistudio.google.com/app/apikey](https://aistudio.google.com/app/apikey)**
2. Sign in with your **Google / Gmail account**
3. Click **"Create API Key"**
4. Copy the key — it looks like `AIzaSy...`

> **No credit card. No charges. Ever.** Google gives 1,500 free requests/day.

### Step 3 — Add key to FitAI

1. Open the app → Sign in
2. A popup will appear asking for your key — paste it and click **"Save & Activate"**
3. Or go to **Profile → API Key section** and paste it there
4. Done! All AI features are now active ✅

## 🤖 AI Features Explained

### Food Search (AI mode)
Type any food in natural language — Gemini AI returns:
- Calories, Protein, Carbs, Fats, Fiber, Sugar, Sodium
- A health note about the food
- One-click "Add to log"

Examples that work:
```
2 chapati with ghee
100g paneer
1 bowl dal tadka
1 large banana
2 boiled eggs
1 scoop whey protein 30g
```

### AI Analysis
Click any **"✦ Analyze"** button to get:
- **Daily** — what went well, what to improve, 3 suggestions for tomorrow
- **Weekly** — patterns, trends, what's working
- **Monthly** — full coach review, progress celebration, next month's focus

### AI Meal Planner
Set your goal, cuisine, dietary restrictions and meal count — Gemini builds a full structured meal plan with calories and macros for each meal.

## 🔐 Privacy & Security

- Your Gemini API key is stored **only in your browser's localStorage**
- The key is **never sent to any server** except Google's AI API directly
- No user data is collected or stored anywhere remotely
- All data stays on your device
